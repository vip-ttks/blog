objdump反汇编程序

工作技术 admin 2个月前 (12-08) 22℃

objdump用来反汇编很简单，能够看到很多内容，指令，命令如下

#!/bin/bash
rm -f demo
rm -f code
g++ -o demo main.cpp
objdump -D -s -M intel demo >>code
vi code

现在正式进行反汇编，然后修改二进制代码，进行破解

源代码如下，命名为main.cpp就能用上面的脚本进行反汇编了

#include <stdio.h>
#include <stdlib.h>

char ch[]="0123456789";
int main()
{
        printf("%s\n",ch);
        return 0;
}

这个程序功能也很简单，就是打印字符串0123456789

我们的破解目标是打印字符串abcdefghi

有两个方法，方法一就是直接g更改二进制数据0123456789为abcdefghi 这里就不说这种方法了

说方法二，添加新的变量 然后赋值为abcdefghi，然后把printf传参改成这个

步骤一：添加新变量

反汇编后的关键代码

图中0x0920的一行本来全部是0的，被我更改成字符串abcdefghi了
