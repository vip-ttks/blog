lua使用管道获取结果

工作技术 admin 6个月前 (10-09) 28℃

lua调用wget,curl等命令的时候可以使用os.excute，但是如果想要获取执行结果的话这个函数就不行了

可以用管道io.popen下面是一个实例

function get(url)
    local handle=io.popen("curl -q -k -s -m 1 "..url)
    local ret=handle:read("*a")
    handle:close()
    return ret
end
