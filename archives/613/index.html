hiredis编译静态版本

工作技术 admin 2个月前 (03-15) 8℃

hiredis是redis的C++客户端，通常make后产生的是libhiredis.so.0.11文件，如果想要生成静态库就这样编译

#!/bin/bash
cur=$(pwd)
echo $cur
export PREFIX=$cur/bin
rm -rf $cur/bin
mkdir $cur/bin
make clean
make install
rm -f $cur/bin/lib/libhiredis.so*
ls -lh $cur/bin/lib
