gearman环境搭建

工作技术 admin 1年前 (2017-04-12) 121℃

公司的转码服务是通过gearman来运转的，之前运维辞职了居然就跑不起来了。

在Ubuntu 14.04.5 LTS的基础上安装gearman，完成调试

1.修改源

由于默认是英文是美国的源比较慢，需要改成中文源，运行脚本source.sh 内容如下

#!/bin/bash
sudo cp /etc/apt/sources.list /etc/apt/sources.list.old
sudo rm -f s.list
echo "deb http://mirrors.sohu.com/ubuntu/ trusty main restricted universe multiverse
deb http://mirrors.sohu.com/ubuntu/ trusty-security main restricted universe multiverse
deb http://mirrors.sohu.com/ubuntu/ trusty-updates main restricted universe multiverse
deb http://mirrors.sohu.com/ubuntu/ trusty-proposed main restricted universe multiverse
deb http://mirrors.sohu.com/ubuntu/ trusty-backports main restricted universe multiverse
deb-src http://mirrors.sohu.com/ubuntu/ trusty main restricted universe multiverse
deb-src http://mirrors.sohu.com/ubuntu/ trusty-security main restricted universe multiverse
deb-src http://mirrors.sohu.com/ubuntu/ trusty-updates main restricted universe multiverse
deb-src http://mirrors.sohu.com/ubuntu/ trusty-proposed main restricted universe multiverse
deb-src http://mirrors.sohu.com/ubuntu/ trusty-backports main restricted universe multiverse">>s.list
sudo mv s.list /etc/apt/sources.list
sudo apt-get update

2.安装，运行gearman，PHP

#!/bin/bash
sudo apt-get install software-properties-common
#sudo add-apt-repository ppa:gearman-developers/ppa
#sudo apt-get update

sudo apt-get install gearman-job-server libgearman-dev -y
sudo apt-get install gearman-tools -y

sudo apt-get install php-pear -y
sudo apt-get install php5-dev -y
sudo pecl install gearman

第一次运行完脚本后 需要手工配置PHP /etc/php5/cli/php.ini在Dynamic Extensions里加上extension=gearman.so

当然也可以直接用脚本

#!/bin/bash
sudo rm -f /etc/php5/cli/php.ini
cd /etc/php5/cli/
sudo wget http://7xl66y.com1.z0.glb.clouddn.com/php.ini

3.测试gearman是否可用

运行好我们的trans转码程序，注意配置文件显式写明端口号，gearman安装完的默认端口是4730,推荐使用下面的配置：

{
"gearman":"127.0.0.1:4730",
"maxrun":1,
"srcprefix":"/data/micro_ticket/src",
"dstprefix":"/data/micro_ticket/dst"
}

现在运行php client.php提交一个测试任务，client.php内容如下：

<?php
$client=new GearmanClient();
$client->addServer();
print $client->doBackground("trans","Hello World!just a test by leisiyuan");
?>

运行完后在日志文件中会出现下面的日志

14:44:59 1491984074087 2017/04/12 16:01:15	json:Hello World!just a test by leisiyuan
14:44:59 1491984074087 2017/04/12 16:01:15	TransFromJson back:{"result":"false","error":"not json string"}
