libxl读取Excel文件的日期

工作技术 admin 4周前 (02-11) 7℃

之前简单的以为libxl的readStr能读取日期，刚好真的读取成功了，例如

2017-12-22

读取出来的字符串就是2017-12-22,交给测试人员测试OK，就没有管了，结果提交给客户的时候直接崩溃了，最终查到是读取日期错误，例如

2018-2-10

这样readStr函数就返回空了，郁闷，后来经过一系列查找才知道原因，真正读取日期的方法是，先判断是否是日期，然后通过读取数字的函数readNum读出数字，再组成年月日，下面是代码

CString key;
if(sheet->isDate(index, 0))
{
	double db = sheet->readNum(index, 0);
	int year=0, month=0, day=0;
	book->dateUnpack(db, &year, &month, &day);
	char tmp[512]={0};
	sprintf(tmp,"%d-%d-%d",year,month,day);
	key=tmp;
	count=0;
}
else
{
	const wchar_t* s = sheet->readStr(index, 0);
	if(!s)
	{
		int tmp=sheet->readNum(index,0);
		++count;
		continue;
	}
	else count=0;
	key=s;
}

这样读取出来的字符串就保存到key中了
