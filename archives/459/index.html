ffmpeg转换视频为pcm数据

未分类 admin 3个月前 (11-11) 16℃

ffmpeg转换音频，视频文件为指定格式的pcm文件，方便其他地方需要用到pcm数据，命令如下

ffmpeg -i d:/Media/a.ts -vn -acodec pcm_f32le -ar 44100 -ac 2 -f f32le d:/test.pcm

转换16bits的格式，可以提供给wasapi使用

ffmpeg -i d:/Media/a.ts -vn -acodec pcm_s16le -ar 44100 -ac 2 -f s16le d:/test.pcm
