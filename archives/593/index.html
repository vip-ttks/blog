centos5.8构建本地yum源

C/C++基础 admin 4小时前 8℃

centos5.8的公网源已经没有了，要想安装一些软件只有自己构建源了，下面是centos5.8的构建过程

下载Iso文件

CentOS-5.8-x86_64-bin-DVD-1of2.iso

如果下载不了的话 可以百度一下下载地址

挂载镜像文件

mount -o loop CentOS-5.8-x86_64-bin-DVD-1of2.iso /mnt/

编写yum配置文件

vi /etc/yum.repos.d/CentOS-Base.repo

先删除里面的所有语句，再输入以下语句

[base]
name=CentOS-$releasever - Base
baseurl=file:///mnt/
gpgcheck=0
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-5

刷新yum缓存

执行以下语句

yum clean all
yum makecache

就大功告成了

中途出了一个小意外就是，我自己之前创建了一个名叫CentOS6-Base-163.repo的文件 导致刷新yum缓存错误，后来删掉了就正常了
