bool的理解

C/C++基础 admin 2个月前 (02-22) 13℃

虽然bool只有true和false两种状态，但是占用的大小是一个字节，并且可以看做是unsigned char，只有是非零就表示true，否则才表示false,下面是代码说明

int main()
{
	bool demo;
	unsigned char* p=(unsigned char*)(&demo);
	unsigned char ch=0xff;
	while(demo)
	{
		*p=ch--;
	}
	printf("%x\n",*p);
	system("pause");
    return 0;

代码执行的结果是0
