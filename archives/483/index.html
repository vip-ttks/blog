不要使用inline

工作技术 admin 2个月前 (12-14) 11℃

当你要使用inline的时候，你应该知道你要做什么，但是编译器不是100%会把你的函数内联，导致的后果是你以为内联了，实际没有。这种情况在VS2005到VS2010都有，所以这种情况下还是用宏来代替inline吧 ，至少能得到确切的结果，用宏写的函数就肯定会内联

下面的我的C代码

inline void demo(char* info){printf("%c",info[0]);}
int main()
{
	{
	demo("lei");
	}
	{
	demo("si");
	}
	{
	demo("yuan");
	}
	system("pause");
	return 0;
}
