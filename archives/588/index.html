redis-cli查看redis数据

工作技术 admin 3个月前 (03-06) 19℃

有时候和redis交互的时候，需要查看数据是否在redis中，可以直接用redis-cli进行查看

安装redis-cli

用yum命令安装

yum -y install redis

连接redis

通常用到两个参数-h 后面跟IP；-p 后面跟端口

例如连接本地redis

redis-cli -h 127.0.0.1 -p 6379

查看所有KEY

keys *

删除某个key

例如删除一个叫demo的key

del demo

查看某个hash的所有key 例如该hash名为msg-node

hkeys msg-node
